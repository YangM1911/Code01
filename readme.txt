----------------
How to use
----------------
The code is tested in MATLAB 2020a(64bit) under the MS Windows 10 64bit version with an AMD Ryzen 7 4800H with Radeon Graphics CPU and 16 GB RAM.

1. unpack the package
2. include code/subdirectory in your Matlab path
3. Run "demo.m" to try the examples included in this package.
